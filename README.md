#  Number Guessing Game (C++)

##  Description
This is a simple console-based Number Guessing Game written in C++.  
The program generates a random number between 1 and 100, and the user has to guess it.

---

##  Features
-  Random number generation (1–100)
-  Continuous guessing using loop
-  Hints for high and low guesses
-  Winning message on correct guess

---

## Technologies Used
- C++
- Standard Library (`iostream`, `cstdlib`, `ctime`)

---
